# Flask Docker Kubernetes Project

This project demonstrates a simple Flask application
containerized using Docker and deployed on Kubernetes
using Minikube.

## Tech Stack
- Python (Flask)
- Docker
- Kubernetes (Minikube)

## Steps to Run
1. Build Docker image
2. Deploy using Kubernetes manifests
3. Expose service using NodePort
4. Access application via browser

## Output
Application runs on port 5000 and shows a success message.
